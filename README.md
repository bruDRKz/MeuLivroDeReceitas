# 🍽️ API de Receitas (.NET)

🚧 **Projeto em desenvolvimento** 🚧

Este projeto é uma **API de receitas**, desenvolvida como parte de um curso focado em criação de APIs reais com .NET.

O objetivo é aplicar, na prática, conceitos e boas práticas usadas em projetos profissionais, como arquitetura limpa, DDD, testes, CI/CD e deploy em cloud.

O código pode sofrer mudanças frequentes, refatorações e ajustes conforme o andamento do curso e do aprendizado.

Este repositório tem caráter **educacional** e **não está pronto para produção** no momento.
